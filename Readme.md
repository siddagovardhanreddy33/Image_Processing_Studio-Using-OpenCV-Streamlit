# 📷 Image Preprocessor using OpenCV & Streamlit

An interactive image preprocessing and augmentation application built using Python, OpenCV, and Streamlit. The project provides commonly used image enhancement and preprocessing operations used in Computer Vision and CNN pipelines.

## 🚀 Features

### Image Enhancement

* Brightness Adjustment
* Contrast Adjustment
* Sharpening

### Image Filters

* Grayscale
* Gaussian Blur
* Thresholding
* Negative Effect
* Sepia Filter

### Edge Detection

* Canny Edge Detection

### Image Augmentation

* Horizontal Flip
* Vertical Flip
* Rotation
* Noise Addition

### CNN Preprocessing

* Image Resizing
* Pixel Normalization (0-1)

### Output

* Original vs Processed Image Comparison
* Download Processed Image

## 🛠️ Tech Stack

* Python
* OpenCV
* Streamlit
* NumPy
* Pillow

```bash
git clone <repository-url>

cd image_preprocessor

pip install -r requirements.txt

streamlit run app.py
```
